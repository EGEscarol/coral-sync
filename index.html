Configurar Almacenamiento en la Nube

Para que los datos sean accesibles desde cualquier dispositivo, necesitas una cuenta gratuita en JSONBin.io.

Pasos (solo una vez):
1. Ve a jsonbin.io y crea una cuenta gratuita
2. En tu dashboard ve a API Keys
3. Copia tu Secret Key (empieza con $2a$)
4. Pégala aquí abajo y presiona "Crear Base de Datos"
No se pudo conectar. Verifica que la key sea correcta.

JSONBin.io es gratuito y no requiere tarjeta de crédito.
